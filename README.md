# Amazon-Web-Scraping
# 🛒 Amazon Price Monitor using Python

## 📌 Project Overview

This project is an automated Amazon Price Tracker built with Python. It scrapes product information from Amazon, extracts the product title, price, and current date, and stores the data in a CSV file. The collected data can be used to monitor price changes over time and can be extended to send email notifications when the price drops below a desired value.

---

## 🚀 Features

- Scrapes Amazon product details
- Extracts product title
- Extracts current product price
- Records the current date
- Stores data into a CSV file
- Tracks price history over multiple runs
- Easy to extend with email alerts and dashboards

---

## 🛠️ Technologies Used

- Python
- Requests
- BeautifulSoup4
- Pandas
- CSV
- Datetime
- Jupyter Notebook
